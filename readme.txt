== Frequently Asked Questions ==

= What is the purpose of this plugin? =

This plugin allow you to put on your website latest news from Romania Libera (http://www.romanialibera.ro) a national newspaper from Romania.

= What version number means? =

The version number show you latest releases of this plugin. Try keep it up to date.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `romania-libera-rss` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget generated in any sidebar from your website.
4. Enjoy!
